## ✨ Features

### 📱 1. Browse Apps

* 📦 View all apps in a clean responsive grid
* 🔍 Live search functionality for quick access
* 🔽 Filter and sort apps easily

---

### 📊 2. App Details

* 📄 Detailed information about each app
* ⭐ Ratings & reviews with visual charts (Recharts)
* 📥 Download count and app statistics

---

### 📥 3. Installation Management

* ⚡ One-click app installation
* 📂 Track installed apps
* ❌ Uninstall anytime
* 🧾 View installation history

---

### 💾 4. Local Storage

* 🔐 Data stored locally (no backend needed)
* ♻️ Persistent installations after reload
* ⚙️ Automatic data sync

---

### 📱 5. Responsive Design

* 📲 Mobile-first design
* 💻 Optimized for tablet & desktop
* 🌐 Cross-browser compatible

---

### 🎯 6. User Experience

* ⏳ Smooth loading animations
* 🔔 Toast notifications for actions
* 🧭 Easy navigation with active routes
* ⚠️ Custom error handling & messages

---

## 🔮 Upcoming Features

## 🛠️ Technologies Used

## 📌 Installation Guide

```bash
# 📥 Clone the repository
git clone https://github.com/your-username/your-repo-name.git

# 📂 Navigate to project folder
cd your-repo-name

# 📦 Install dependencies
npm install

# ▶️ Run the development server
npm run dev
```

---

🌐 Visit 👉 `http://localhost:5173` in your browser to view the app.
